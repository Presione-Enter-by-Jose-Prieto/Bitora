# Sistema de gestión y seguimiento de etapa productiva para aprendices SENA - Bitora
La plataforma es un sistema web diseñado para gestionar y monitorear el proceso de etapa productiva de los aprendices SENA, permitiendo centralizar el seguimiento académico y administrativo de manera organizada y eficiente. El sistema cuenta con módulos para aprendices, instructores y administradores, donde cada usuario accede a funcionalidades específicas según su rol. Los aprendices pueden registrar bitácoras de avance, consultar información relacionada con su proceso y mantener comunicación directa con el instructor encargado del seguimiento de su ficha. Los instructores disponen de paneles de control para visualizar las fichas asignadas, acceder al listado de aprendices, revisar bitácoras, gestionar el seguimiento y controlar el estado de visitas en los casos correspondientes a modalidad de contrato. Por su parte, el administrador es responsable de la gestión organizacional del sistema, incluyendo el registro de instructores, creación de fichas y asignación de responsables para el monitoreo. La plataforma busca optimizar el control de la etapa productiva mediante una estructura centralizada, trazable y orientada al seguimiento continuo del proceso formativo.
